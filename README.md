# Supplementary repository

This data-free repository contains the LaTeX source and compiled four-page main manuscript, six-page supplementary material, the frozen five-configuration contract, ontology, raw-cache manifest and SHA-256 values, coverage audits, V4 derived records, vector figures, and evaluator/plotting scripts. Raw images, trained weights, and raw prediction caches are not redistributed because they are licensed or large external artifacts; their paths and hashes remain in `outputs/candidates/prediction_manifest_v2.csv`.

The main analysis is intentionally ordered as support mismatch, controlled support perturbation, metric uncertainty, and common-coverage rank evidence. Reproduction order: verify `SHA256SUMS`; inspect raw coverage and `configs/resubmission_visdrone_controlled_pool.yaml`; verify `label_reliability/support.py`; inspect the V4 rule grid and image-paired bootstrap rows; then regenerate the figures with the included scripts. An anonymous reviewer mirror is available at https://anonymous.4open.science/r/aerial-evaluation-audit-5565/README.md.

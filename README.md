# Supplementary repository

This data-free bundle contains the compiled five-page manuscript and six-page supplementary material, frozen rule contract, ontology, raw-cache manifest and SHA-256 values, coverage audits, corrected derived records, pairwise-policy diagnostics, figure-source records, vector figures, and evaluator/plotting scripts. It also includes two standalone 88-mm empirical composite figures (A and B), their source records, and the deterministic rendering script; the requested workflow/architecture panel is intentionally omitted. Raw images and prediction-cache Parquet files are not redistributed: they are licensed external artifacts and their paths/hashes are recorded in `outputs/candidates/prediction_manifest_v2.csv`.

The anonymous supplementary repository is https://anonymous.4open.science/r/aerial-evaluation-audit-5565/README.md. Reproduction order: verify `SHA256SUMS`; inspect coverage and the frozen candidate manifest; evaluate AP and the corrected absolute/normalized grids; run the Hungarian, removed-as-ignore, paired-bootstrap, and pairwise-policy controls.

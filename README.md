# Supplementary repository

This data-free repository contains the LaTeX source and compiled five-page main manuscript, four-page supplementary material, frozen configuration and ontology records, derived audit outputs, vector figures, evaluator code, plotting scripts, tests, and SHA-256 checksums. Raw images, trained weights, and raw prediction caches are not redistributed because they are licensed or large external artifacts; their declared locations and hashes remain in the released manifests and provenance tables.

The audit order is: (0) qualify prediction coverage; (1) measure full-annotation support; (2) replay the declared A→F→R→S contract path and verify its TP/FP/FN ledger across frozen configurations; and (3) compare candidates only on common coverage with fixed-F1, max-F1, AP, and simultaneous sequence-cluster intervals. The full-coverage path cohort contains five VisDrone and four UAVDT configurations; three AI-TOD resolution slices remain coverage-conditioned controls. A named-pair 2×2 control crosses valid-match IoU {.25,.50} with threshold-optimized F1 and score-integrated AP.

Start by verifying `SHA256SUMS`. The primary derived records are in `outputs/coverage_qualification/`, `outputs/cross_configuration_contract_paths/`, `outputs/visdrone_path_support_robustness/`, and `outputs/metric_factorial_control/`. The corresponding replay scripts and figure builders are included. Exploratory full-grid and matching-control records remain separate from the confirmatory simultaneous family.

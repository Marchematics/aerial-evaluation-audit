# Supplementary repository

This data-free repository contains the LaTeX source and compiled five-page main manuscript, three-page supplementary material, frozen configuration and ontology records, derived audit outputs, vector figures, evaluator code, plotting scripts, tests, and SHA-256 checksums. Raw images, trained weights, and raw prediction caches are not redistributed because they are licensed or large external artifacts; their declared locations and hashes remain in `outputs/candidates/prediction_manifest_v2.csv`.

The audit order is: (0) qualify prediction coverage; (1) measure full-annotation support; (2) replay the declared A→F→R→S contract path and verify its TP/FP/FN ledger; and (3) compare frozen candidates only on common coverage with fixed-F1, max-F1, AP, and paired sequence-cluster intervals. A named-pair 2×2 control crosses valid-match IoU {.25,.50} with threshold-optimized F1 and score-integrated AP. It shows that a sequence-resolved ASD-favoring normalized conclusion appears only when stricter localization and score integration are combined.

Start by verifying `SHA256SUMS`. The primary derived records are in `outputs/coverage_qualification/`, `outputs/contract_path/`, `outputs/metric_qualified_rank/`, and `outputs/metric_factorial_control/`. The main evaluators are `scripts/evaluate_aitod_coverage_qualification.py`, `scripts/evaluate_contract_path_headlines.py`, `scripts/evaluate_metric_qualified_visdrone_rank.py`, and `scripts/evaluate_metric_factorial_control.py`; figure builders are the four included `build_*` scripts. Exploratory full-grid and matching-control records are retained separately and are not used for post-selection pointwise claims.

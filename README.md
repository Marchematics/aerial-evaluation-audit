# Supplementary repository

This repository contains the LaTeX source and compiled five-page main manuscript, four-page supplementary material, frozen configuration and ontology records, derived audit outputs, vector figures, evaluator code, plotting scripts, tests, and SHA-256 checksums. Dataset licenses and artifact size place raw images, trained weights, and prediction caches in their source archives; released manifests and provenance tables provide their locations and hashes.

The audit order is: (0) qualify prediction coverage; (1) measure full-annotation support; (2) replay the declared A→F→R→S contract path and verify its TP/FP/FN ledger across frozen configurations; and (3) compare candidates on common coverage with fixed-F1, max-F1, AP, and simultaneous sequence-cluster intervals. The full-coverage path cohort contains five VisDrone and four UAVDT configurations; three AI-TOD resolution slices provide coverage-conditioned controls. A named-pair 2×2 control crosses valid-match IoU {.25,.50} with threshold-optimized F1 and score-integrated AP.

Start by verifying `SHA256SUMS`. The primary derived records are in `outputs/coverage_qualification/`, `outputs/cross_configuration_contract_paths/`, `outputs/visdrone_path_support_robustness/`, `outputs/visdrone_scene_consequences/`, and `outputs/metric_factorial_control/`. The scene records stratify density, target size, neighbor spacing, and occlusion and report FP/image, FN per 100 scored vehicles, and confirmed vehicles per 100 scored alerts. The corresponding replay scripts and figure builders are included.
